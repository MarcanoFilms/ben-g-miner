# Changelog

## [1.0.0] - 2026-06-24

### Initial Release
- Complete bfgminer setup with DATUM gateway patches
- GPU support for NVIDIA cards (RTX 30-series, 40-series, A-series)
- Automated installation script
- systemd service integration
- Monitoring dashboard compatibility
- Full documentation
- MIT License

### Known Limitations
- GPU mining is not profitable (educational purpose only)
- Shares extremely rare due to network difficulty
- Expected time to find a block: ~9,370 years with RTX 3060 Ti
